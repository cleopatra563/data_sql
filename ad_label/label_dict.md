### 2. 一级标签：阶段和目标
**定义**：回答"这个素材为何存在？从哪来？"——该素材被制作和投放所要达成的核心商业目的，以及面向设计师的阶段指导。

| 二级标签 | 三级标签 | 描述 | 示例 |
|---------|---------|------|------|
| 素材阶段 | 测试期（探索期） | 游戏上线前的测试阶段，帮助设计师找到"跑量的创意方向" | 每周产出10-20条短视频测试方向的素材 |
|  | 预注册期（蓄水期） | 游戏上线前的预约阶段，帮助设计师制造期待感与品牌识别 | 高品质CG+剧情向的预注册宣传素材 |
|  | 首发期（爆发期） | 游戏正式上线的推广阶段，帮助设计师实现短期爆量拉新+转化最大化 | 快节奏+挑战式的首发月推广素材 |
|  | 长线运营期 | 游戏稳定运营阶段，帮助设计师维持买量效率与品牌持续感 | 节日主题+版本更新的长期运营素材 |
| 营销目标 | 拉新 | 吸引新用户下载安装 | 强调"免费下载"的素材 |
|  | 促活 | 促进现有用户活跃 | 强调"新活动"的素材 |
|  | 付费 | 促进用户付费 | 强调"限时礼包"的素材 |
|  | 预约 | 引导用户预约游戏 | 强调"预约奖励"的素材 |
|  | 品牌 | 提升品牌知名度 | 强调游戏世界观和IP的素材 |
|  | 召回 | 召回流失用户 | 强调"回归奖励"的素材 |
| 目标用户 | 核心用户 | 深度理解并热爱游戏题材与玩法，是产品的主要留存与付费群体 | 强调游戏深度内容的素材 |
|  | 精准用户 | 对游戏题材有兴趣，对玩法接受度较高，有潜力转化为核心用户 | 强调游戏题材与核心玩法的素材 |
|  | 泛用户 | 对游戏产品产生一定兴趣，但不属于核心圈层 | 强调游戏趣味性与广泛吸引力的素材 |
|  | 误导用户 | 通过不相关或虚假内容吸引来的用户，实际对产品无兴趣或无法接受 | 利用擦边内容或夸张演绎吸引点击的素材 |
